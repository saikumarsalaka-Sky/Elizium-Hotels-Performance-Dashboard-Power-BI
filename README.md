This project is an interactive Power BI dashboard built for Elizium Hotels to analyze hotel business performance across multiple properties.
It provides deep insights into revenue, occupancy, pricing, booking platforms, customer segments, and operational KPIs to support data-driven decision making.

The dashboard enables management to monitor trends, compare property performance, and identify revenue optimization opportunities.

Weekly trend analysis of key metrics

City

Room Class
